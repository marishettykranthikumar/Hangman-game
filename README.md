# 🎮 Hangman Game

## Description

This is a simple text-based Hangman Game developed using Python. The player guesses a hidden word one letter at a time within a limited number of attempts.

## Features

* Random word selection
* Letter-by-letter guessing
* Tracks incorrect attempts
* Win/Lose conditions
* User-friendly console interface

## Technologies Used

* Python
* Random Module

## Concepts Used

* Loops
* Conditional Statements
* Lists
* Strings
* Functions

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run:

python hangman.py

## Output

The game displays a hidden word using underscores and allows the player to guess letters until the word is completed or attempts run out.
